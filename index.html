AstroTaverna

To install the Alpha AstroTaverna plugin to Taverna:

  • Download and install Taverna 2.4
  • Start Taverna
  • Add a plugin site: http://julian-garrido.github.com/astrotaverna/
  • Restart Taverna
  • The VO services perspective should now appear together with variouys local tools under Available Services

This is an alpha version an the content of this plugin may not be in the official release

For more information, see the Astrotaverna wiki page.
